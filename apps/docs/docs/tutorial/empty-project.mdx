# Empty Project

Now that you have your react project, and `OpenVP` is installed, let's add the `<Environment>` to your app.

```jsx live
function Clock(props) {
  const [date, setDate] = useState(new Date());
  useEffect(() => {
    const timerID = setInterval(() => tick(), 1000);

    return function cleanup() {
      clearInterval(timerID);
    };
  });

  function tick() {
    setDate(new Date());
  }

  return (
    <div>
      <h2>It is {date.toLocaleTimeString()}.</h2>
    </div>
  );
}
```

By default, the `<Environment>` expands to fill the size of the parent element. Since our app covers the full webpage, `OpenVP` is full-screen.
